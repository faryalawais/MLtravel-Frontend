## 7. Accessibility

- **Landmark:** `<header>` wrapping navbar; `nav` for link group (desktop).
- **Labels:** CTA accessible name "Book A Demo"; menu button `aria-label="Open menu"` (mobile).
- **Focus order:** Logo/brand → Product → How It Works → Pricing → Book A Demo.
- **Keyboard:** Links activatable via Enter; CTA via Enter/Space; visible focus ring using `color.focus.ring`.
- **Contrast:** Text and CTA meet WCAG AA against `color.background.page`.
