## Token name reference

All tokens below are available in:
- CSS: `var(--<token-path-with-hyphens>)` e.g. `var(--color-surface-card)`
- Tailwind: via the token class utilities in `tokens/build/tailwind-tokens.js`

All token names must be used **exactly as listed** in contract.md and
implementation. Approximate substitutions (nearest-neighbour) are fidelity bugs.
