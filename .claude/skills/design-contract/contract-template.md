## 2. Component anatomy
> Every named element must carry a `[component.*]  data-testid` tag on its
> line. This is the machine-readable anchor that `validate:contract` parses.
> An element without this tag is invisible to the gate.
> Format:  `├─ <ElementName>  [component.X.Y.Z]  data-testid`
```
<RootComponent>  [component.<feature>.<screen>.root]  data-testid
├─ <Child>  [component.<feature>.<screen>.child]  data-testid — <role>
│  └─ <Grandchild>  [component.<feature>.<screen>.grandchild]  data-testid — <role>
└─ <Child>  [component.<feature>.<screen>.otherChild]  data-testid — <role>
```
